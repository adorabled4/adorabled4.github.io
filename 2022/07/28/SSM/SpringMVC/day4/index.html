1.UUID з®Җд»Ӣ

  • UUID еҗ«д№үжҳҜйҖҡз”Ёе”ҜдёҖиҜҶеҲ«з Ғ (Universally Unique Identifier)пјҢиҝҷжҳҜдёҖдёӘиҪҜд»¶е»әжһ„зҡ„ж ҮеҮҶгҖӮ

    д№ҹжҳҜиў«ејҖжәҗиҪҜд»¶еҹәйҮ‘дјҡ (Open Software Foundation, OSF) зҡ„з»„з»Үеә”з”ЁеңЁеҲҶеёғејҸи®Ўз®—зҺҜеўғ (Distributed Computing Environment, DCE) йўҶеҹҹзҡ„дёҖйғЁеҲҶгҖӮ

    UUID зҡ„зӣ®зҡ„пјҢжҳҜи®©еҲҶеёғејҸзі»з»ҹдёӯзҡ„жүҖжңүе…ғзҙ пјҢйғҪиғҪжңүе”ҜдёҖзҡ„иҫЁиҜҶиө„и®ҜпјҢиҖҢдёҚйңҖиҰҒйҖҸиҝҮдёӯеӨ®жҺ§еҲ¶з«ҜжқҘеҒҡиҫЁиҜҶиө„и®Ҝзҡ„жҢҮе®ҡгҖӮ

    еҰӮжӯӨдёҖжқҘпјҢжҜҸдёӘдәәйғҪеҸҜд»Ҙе»әз«ӢдёҚдёҺе…¶е®ғдәәеҶІзӘҒзҡ„ UUIDгҖӮеңЁиҝҷж ·зҡ„жғ…еҶөдёӢпјҢе°ұдёҚйңҖиҖғиҷ‘ж•°жҚ®еә“е»әз«Ӣж—¶зҡ„еҗҚз§°йҮҚеӨҚй—®йўҳгҖӮ

2.UUID з»„жҲҗ

UUIDдҝқиҜҒеҜ№еңЁеҗҢдёҖж—¶з©әдёӯзҡ„жүҖжңүжңәеҷЁйғҪжҳҜе”ҜдёҖзҡ„гҖӮйҖҡеёёе№іеҸ°дјҡжҸҗдҫӣз”ҹжҲҗзҡ„APIгҖӮ

жҢүз…§ејҖж”ҫиҪҜд»¶еҹәйҮ‘дјҡ(OSF)еҲ¶е®ҡзҡ„ж ҮеҮҶи®Ўз®—пјҢз”ЁеҲ°дәҶд»ҘеӨӘзҪ‘еҚЎең°еқҖгҖҒзәіз§’зә§ж—¶й—ҙгҖҒиҠҜзүҮIDз Ғе’Ңи®ёеӨҡеҸҜиғҪзҡ„ж•°еӯ—гҖӮ

  1. еҪ“еүҚж—Ҙжңҹе’Ңж—¶й—ҙпјҢUUIDзҡ„з¬¬дёҖдёӘйғЁеҲҶдёҺж—¶й—ҙжңүе…іпјҢеҰӮжһңдҪ еңЁз”ҹжҲҗдёҖдёӘUUIDд№ӢеҗҺпјҢиҝҮеҮ з§’еҸҲз”ҹжҲҗдёҖдёӘUUIDпјҢеҲҷз¬¬дёҖдёӘйғЁеҲҶдёҚеҗҢпјҢе…¶дҪҷзӣёеҗҢгҖӮ
  2. ж—¶й’ҹеәҸеҲ—гҖӮ
  3. е…ЁеұҖе”ҜдёҖзҡ„IEEEжңәеҷЁиҜҶеҲ«еҸ·пјҢеҰӮжһңжңүзҪ‘еҚЎпјҢд»ҺзҪ‘еҚЎMACең°еқҖиҺ·еҫ—пјҢжІЎжңүзҪ‘еҚЎд»Ҙе…¶д»–ж–№ејҸиҺ·еҫ—гҖӮUUIDзҡ„е”ҜдёҖзјәйҷ·еңЁдәҺз”ҹжҲҗзҡ„з»“жһңдёІдјҡжҜ”иҫғй•ҝгҖӮе…ідәҺUUIDиҝҷдёӘж ҮеҮҶдҪҝз”ЁжңҖжҷ®йҒҚзҡ„жҳҜеҫ®иҪҜзҡ„GUID(Globals Unique Identifiers)гҖӮ

  • postHandleпјҡжҺ§еҲ¶еҷЁж–№жі•жү§иЎҢд№ӢеҗҺжү§иЎҢpostHandle()

иҝҷдёӘзұ»жҳҜдёәйӮЈдәӣеЈ°жҳҺдәҶпјҲ@ExceptionHandlerгҖҒ@InitBinder жҲ– @ModelAttributeжіЁи§Јдҝ®йҘ°зҡ„пјүж–№жі•зҡ„зұ»иҖҢжҸҗдҫӣзҡ„дё“дёҡеҢ–зҡ„@Component , д»ҘдҫӣеӨҡдёӘ Controllerзұ»жүҖе…ұдә«гҖӮ

иҜҙзҷҪдәҶпјҢе°ұжҳҜaopжҖқжғізҡ„дёҖз§Қе®һзҺ°пјҢдҪ е‘ҠиҜүжҲ‘йңҖиҰҒжӢҰжҲӘи§„еҲҷпјҢжҲ‘её®дҪ жҠҠд»–д»¬жӢҰдёӢжқҘпјҢе…·дҪ“дҪ жғіеҒҡжӣҙз»ҶиҮҙзҡ„жӢҰжҲӘзӯӣйҖүе’ҢжӢҰжҲӘд№ӢеҗҺзҡ„еӨ„зҗҶпјҢдҪ иҮӘе·ұйҖҡиҝҮ@ExceptionHandlerгҖҒ@InitBinder жҲ– @ModelAttributeиҝҷдёүдёӘжіЁи§Јд»ҘеҸҠиў«е…¶жіЁи§Јзҡ„ж–№жі•жқҘиҮӘе®ҡд№үгҖӮ

  • йҰ–е…ҲпјҢControllerAdviceжң¬иҙЁдёҠжҳҜдёҖдёӘComponentпјҢеӣ жӯӨд№ҹдјҡиў«еҪ“жҲҗз»„е»әжү«жҸҸпјҢдёҖи§ҶеҗҢд»ҒпјҢжү«жү«жү«гҖӮ

еҲқе®ҡд№үжӢҰжҲӘи§„еҲҷпјҡ

  • ControllerAdvice жҸҗдҫӣдәҶеӨҡз§ҚжҢҮе®ҡAdviceи§„еҲҷзҡ„е®ҡд№үж–№ејҸпјҢй»ҳи®Өд»Җд№ҲйғҪдёҚеҶҷпјҢеҲҷжҳҜAdviceжүҖжңүControllerпјҢеҪ“з„¶дҪ д№ҹеҸҜд»ҘйҖҡиҝҮдёӢеҲ—зҡ„ж–№ејҸжҢҮе®ҡи§„еҲҷ
    жҜ”еҰӮеҜ№дәҺ String[] value() default {} , еҶҷжҲҗ@ControllerAdvice("org.my.pkg") жҲ–иҖ… @ControllerAdvice(basePackages="org.my.pkg"), еҲҷеҢ№й…Қorg.my.pkgеҢ…еҸҠе…¶еӯҗеҢ…дёӢзҡ„жүҖжңүControllerпјҢ

  • еҪ“з„¶д№ҹеҸҜд»Ҙз”Ёж•°з»„зҡ„еҪўејҸжҢҮе®ҡпјҢеҰӮпјҡ@ControllerAdvice(basePackages={"org.my.pkg", "org.my.other.pkg"}), д№ҹеҸҜд»ҘйҖҡиҝҮжҢҮе®ҡжіЁи§ЈжқҘеҢ№й…ҚпјҢжҜ”еҰӮжҲ‘иҮӘе®ҡдәҶдёҖдёӘ @CustomAnnotation жіЁи§ЈпјҢжҲ‘жғіеҢ№й…ҚжүҖжңүиў«иҝҷдёӘжіЁи§Јдҝ®йҘ°зҡ„ Controller, еҸҜд»Ҙиҝҷд№ҲеҶҷпјҡ

    @ControllerAdviceпјҲannotations={CustomAnnotation.class})

жәҗз Ғ

1.еӨ„зҗҶе…ЁеұҖејӮеёё

@ControllerAdvice й…ҚеҗҲ @ExceptionHandler е®һзҺ°е…ЁеұҖејӮеёёеӨ„зҗҶ

  • з”ЁдәҺеңЁзү№е®ҡзҡ„еӨ„зҗҶеҷЁзұ»гҖҒж–№жі•дёӯеӨ„зҗҶејӮеёёзҡ„жіЁи§Ј

жҺҘж”¶Throwableзұ»дҪңдёәеҸӮж•°пјҢжҲ‘д»¬зҹҘйҒ“ThrowableжҳҜжүҖжңүејӮеёёзҡ„зҲ¶зұ»пјҢжүҖд»ҘиҜҙпјҢеҸҜд»ҘиҮӘиЎҢжҢҮе®ҡжүҖжңүејӮеёёжҜ”еҰӮеңЁж–№жі•дёҠеҠ пјҡ@ExceptionHandler(IllegalArgumentException.class)пјҢ

еҲҷиЎЁжҳҺжӯӨж–№жі•еӨ„зҗҶIllegalArgumentException зұ»еһӢзҡ„ејӮеёёпјҢ

еҰӮжһңеҸӮж•°дёәз©әпјҢе°Ҷй»ҳи®Өдёәж–№жі•еҸӮж•°еҲ—иЎЁдёӯеҲ—еҮәзҡ„д»»дҪ•ејӮеёёпјҲж–№жі•жҠӣеҮәд»Җд№ҲејӮеёёйғҪжҺҘеҫ—дҪҸпјүгҖӮ

дёӢйқўзҡ„дҫӢеӯҗпјҡ

  • еӨ„зҗҶжүҖжңүIllegalArgumentExceptionејӮеёёпјҢеҹҹдёӯеҠ е…Ҙй”ҷиҜҜдҝЎжҒҜerrorMessage е№¶иҝ”еӣһй”ҷиҜҜйЎөйқўerror

2.йў„и®ҫе…ЁеұҖж•°жҚ®

@ControllerAdvice й…ҚеҗҲ @ModelAttribute йў„и®ҫе…ЁеұҖж•°жҚ®

@ModelAttributeжәҗз Ғ

  • е®һйҷ…дёҠиҝҷдёӘжіЁи§Јзҡ„дҪңз”Ёе°ұжҳҜпјҢе…Ғи®ёдҪ еҫҖ Model дёӯжіЁе…Ҙе…ЁеұҖеұһжҖ§пјҲеҸҜд»ҘдҫӣжүҖжңүControllerдёӯжіЁжңү@Request Mappingзҡ„ж–№жі•дҪҝз”ЁпјүпјҢvalue е’Ң name з”ЁдәҺжҢҮе®ҡ еұһжҖ§зҡ„ key пјҢbinding иЎЁзӨәжҳҜеҗҰз»‘е®ҡпјҢй»ҳи®Өдёә trueгҖӮ4
е…·дҪ“дҪҝз”Ёж–№жі•

  • е…ЁеұҖеҸӮж•°з»‘е®ҡ

    • ж–№ејҸдёҖпјҡ

    иҝҷз§Қж–№ејҸжҜ”иҫғзҒөжҙ»пјҢйңҖиҰҒд»Җд№ҲиҮӘе·ұеҠ е°ұиЎҢдәҶпјҢеҠ еӨҡе°‘еұһжҖ§иҮӘе·ұжҺ§еҲ¶

    • ж–№ејҸдәҢпјҡ

    иҝҷз§Қж–№ејҸеҜ№дәҺеҠ еҚ•дёӘеұһжҖ§жҜ”иҫғж–№дҫҝгҖӮй»ҳи®ӨдјҡжҠҠиҝ”еӣһеҖјпјҲеҰӮдёҠйқўзҡ„mapпјүдҪңдёәеұһжҖ§зҡ„valueпјҢиҖҢеҜ№дәҺkeyжңүдёӨз§ҚжҢҮе®ҡж–№ејҸпјҡ

    1. еҪ“ @ModelAttribute() дёҚдј д»»дҪ•еҸӮж•°зҡ„ж—¶еҖҷпјҢй»ҳи®ӨдјҡжҠҠиҝ”еӣһеҖјзҡ„еӯ—з¬ҰдёІеҖјдҪңдёәkeyпјҢеҰӮдёҠдҫӢзҡ„ key еҲҷжҳҜ вҖқmap"пјҲеҖјеҫ—жіЁж„Ҹзҡ„жҳҜпјҢдёҚж”ҜжҢҒеӯ—з¬ҰдёІзҡ„иҝ”еӣһеҖјдҪңдёәkeyпјүгҖӮ
    2. еҪ“ @ModelAttribute("myMap") дј еҸӮж•°зҡ„ж—¶еҖҷпјҢеҲҷд»ҘеҸӮж•°еҖјдҪңдёәkeyпјҢиҝҷйҮҢ key еҲҷжҳҜ вҖқmyMapвҖңгҖӮ

  • е…ЁеұҖеҸӮж•°дҪҝз”Ё

    иҝҷдёүз§Қж–№ејҸеӨ§еҗҢе°ҸејӮпјҢе…¶е®һйғҪжҳҜйғҪжҳҜд»ҺModel дёӯеӯҳеӮЁеұһжҖ§зҡ„ MapйҮҢеҸ–ж•°жҚ®гҖӮ

3.иҜ·жұӮеҸӮж•°йў„еӨ„зҗҶ

@ControllerAdvice й…ҚеҗҲ @InitBinder е®һзҺ°еҜ№иҜ·жұӮеҸӮж•°зҡ„йў„еӨ„зҗҶ

еҶҚж¬Ўд№ӢеүҚжҲ‘д»¬е…ҲжқҘдәҶи§ЈдёҖдёӢ @IniiBinderпјҢе…ҲзңӢдёҖдёӢжәҗз ҒпјҢжҲ‘дјҡжҸҗеҸ–дёҖдәӣйҮҚиҰҒзҡ„жіЁйҮҠиҝӣиЎҢжө…жһҗ

жҲ‘д»¬жқҘзңӢзңӢе…·дҪ“з”ЁйҖ”пјҢе…¶е®һиҝҷдәӣз”ЁйҖ”еңЁ ControllerйҮҢд№ҹеҸҜд»Ҙе®ҡд№үпјҢдҪҶжҳҜдҪңз”ЁиҢғеӣҙе°ұеҸӘйҷҗеҪ“еүҚControllerпјҢеӣ жӯӨдёӢйқўзҡ„дҫӢеӯҗжҲ‘д»¬е°Ҷз»“еҗҲ ControllerAdvice дҪңе…ЁеұҖеӨ„зҗҶгҖӮ

  • еҸӮж•°еӨ„зҗҶ

    иҝҷж ·д№ӢеҗҺе‘ўпјҢе°ұеҸҜд»Ҙе®һзҺ°е…ЁеұҖзҡ„е®һзҺ°еҜ№ Controller дёӯRequestMappingж ҮиҜҶзҡ„ж–№жі•дёӯзҡ„жүҖжңү String е’ҢDateзұ»еһӢзҡ„еҸӮж•°йғҪдјҡиў«дҪңзӣёеә”зҡ„еӨ„зҗҶгҖӮ

    Controllerпјҡ

    жөӢиҜ•з»“жһңпјҡ


жҲ‘д»¬еҸҜд»ҘзңӢеҮәпјҢstr е’Ң date иҝҷдёӨдёӘеҸӮж•°еңЁиҝӣе…Ҙ Controller зҡ„testзҡ„ж–№жі•д№ӢеүҚе·Із»Ҹиў«еӨ„зҗҶдәҶпјҢstr иў«еҺ»жҺүдәҶдёӨиҫ№зҡ„з©әж ј(%20 еңЁHttp url дёӯжҳҜз©әж јзҡ„ж„ҸжҖқ)пјҢStringзұ»еһӢзҡ„ 1997-1-10иў«иҪ¬жҚўжҲҗдәҶDateзұ»еһӢгҖӮ

  • еҸӮж•°з»‘е®ҡ

    еҸӮж•°з»‘е®ҡеҸҜд»Ҙи§ЈеҶізү№е®ҡй—®йўҳпјҢйӮЈд№ҲжҲ‘д»¬е…ҲжқҘзңӢзңӢжҲ‘д»¬йқўдёҙзҡ„й—®йўҳ

    жҲ‘д»¬дјҡеҸ‘зҺ° Personзұ»е’Ң Book зұ»йғҪжңү nameеұһжҖ§пјҢйӮЈд№ҲиҝҷдёӘж—¶еҖҷе°ұдјҡеҮәе…Ҳй—®йўҳпјҢе®ғеҸҜжІЎжңүйӮЈд№ҲеҸӘиғҪеҢәеҲҶе“ӘдёӘnameжҳҜе“ӘдёӘзұ»зҡ„гҖӮеӣ жӯӨ @InitBinderе°ұжҙҫдёҠз”ЁеңәдәҶпјҡ

    еӣ жӯӨпјҢдј е…Ҙзҡ„еҗҢеҗҚдҝЎжҒҜе°ұиғҪеҜ№еә”з»‘е®ҡеҲ°зӣёеә”зҡ„е®һдҪ“зұ»дёӯпјҡ

    p.name -> Person.name b.name -> Book.name

    иҝҳжңүдёҖзӮ№жіЁж„Ҹзҡ„жҳҜеҰӮжһң @InitBinder("value") дёӯзҡ„ value еҖје’Ң Controller дёӯ @RequestMapping() ж ҮиҜҶзҡ„ж–№жі•зҡ„еҸӮж•°еҗҚдёҚеҢ№й…ҚпјҢеҲҷе°ұдјҡдә§з”ҹз»‘е®ҡеӨұиҙҘзҡ„еҗҺжһң,еҰӮпјҡ

    • @InitBinder(вҖңpвҖқ)гҖҒ

    • @InitBinder(вҖңbвҖқ)

    • public void test(Person person, Book book)

    дёҠиҝ°жғ…еҶөе°ұдјҡеҮәзҺ°з»‘е®ҡеӨұиҙҘпјҢжңүдёӨз§Қи§ЈеҶіеҠһжі•пјҡ

    1. з»ҹдёҖеҗҚз§°пјҢиҰҒд№Ҳе…ЁеҸ«pпјҢиҰҒд№Ҳе…ЁеҸ«personпјҢеҸӘиҰҒзӣёеҗҢе°ұиЎҢгҖӮ

    2. ж–№жі•еҸӮж•°еҠ  @ModelAttributeпјҢжңүзӮ№зұ»дјј@RequestParam

  • зүҲжқғеЈ°жҳҺпјҡ жң¬еҚҡе®ўжүҖжңүж–Үз« йҷӨзү№еҲ«еЈ°жҳҺеӨ–пјҢи‘—дҪңжқғеҪ’дҪңиҖ…жүҖжңүгҖӮиҪ¬иҪҪиҜ·жіЁжҳҺеҮәеӨ„пјҒ
